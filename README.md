Вывод значения переменых и их присваивание в консоль
```

{
    val a: Int      //Создание переменной
    a=10            //Присвоение величины к переменной 
    val s = 32      //Можно присвоить и без написания типа даных
   
    println(a)
    println(s)
}
```
